# MERN Grammar Test Application

This project is a full-stack MERN application for taking a grammar test, comparing answers using string-similarity.

## Backend Setup

```bash
cd backend
npm install
npm run dev   # starts nodemon server on port 5000
```
Note: The `.env` file containing the MongoDB connection string (`MONGO_URI`) and `JWT_SECRET` is already configured in the `backend/` directory.

## Frontend Setup

```bash
cd frontend
npm install
npm run dev   # starts Vite dev server
```

The frontend runs on the standard Vite port (usually 5173). Navigate to it in your browser.
